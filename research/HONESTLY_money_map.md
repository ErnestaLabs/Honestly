# Honestly - money map

## Thesis
We do not make money from one paywall. We make money from a stack:
free trust -> paid decision help -> compare-and-connect -> recurring monitoring -> pro / white-label.

## The goldmine
Frame downstream services as a **compare options service**:
- show 3-5 winners for a need
- let the user pick the winner
- route the lead only to the selected winner
- charge the winner for the intro

That keeps the user in control and makes the fee feel like a utility, not a sales scam.

## Compare service pricing bands
Starting bands, tuned by category and intent:
- mortgage broker: £40-120 per qualified intro
- solicitor / conveyancer: £50-150 per intro
- surveyor: £75-250 per intro
- insurance: £10-40 recurring or per-bind
- removals: £20-80 per lead
- utilities / broadband: £10-50 per conversion
- staging / cleaning / trades: £25-150 per job

The user pays nothing. The winner pays for the intro.

## How to get to the £5/day target
A single user does not need to pay us £5 every day in one shot.
We average toward it through the stack:

1. Free appraisal captures the user.
2. The compare service monetises immediate needs.
3. Alerts / watchlists monetise continued intent.
4. Monthly refresh keeps recurring value alive.
5. Pro / white-label monetises heavy users.
6. Partner intros monetize the moment of choice.

For an active user, the realistic goal is:
- consumer: £15-80 total in a decision window
- investor: £30-120/month if they track multiple properties
- agent/pro: £100-600/month
- need-based compare lead: £20-250 per routed win

## Blog weaponization
The blog is not the product. It is the authority engine.

Use it to:
- win SEO / AEO for high-intent questions
- establish trust before Telegram
- route readers into the bot or compare service
- capture email and Telegram leads
- feed analytics on what converts
- feed content ideas from VOC and competitor scans

### Blog roles
1. **Authority** - "we know the market"
2. **Discovery** - answer the question people are already searching
3. **Routing** - push to Telegram / Mini App / compare service
4. **Capture** - email, Telegram start, lead form
5. **Upsell** - alerts, compare packs, Pro, referrals
6. **Analytics** - which posts drive money, not just traffic

### Blog clusters from the VOC
- instant UK home valuation
- mortgage-ready comparison
- renovation ROI / value uplift
- neighbourhood market trends
- legal and tax implications
- why is this house not selling
- down-valued property
- tools to help compare houses
- Rightmove alerts / instant alerts
- home affordability spreadsheet

### CTA order from the Hit pass
1. Start your free valuation
2. See how your value stacks up
3. Download full comparison guide
4. Book a call only on high-value clusters
5. Join community or watchlist if the user is high intent

Each post gets one primary CTA:
- run the free valuation
- compare options
- start a watchlist
- get alerts
- request a recommendation

## Hit usage
Hit is **not** part of Honestly. It is an external tool the user runs across projects.

In this project, Hit is offered only as the **"what people say"** enrichment layer:
- run a scan for market sentiment
- pull exact user language
- surface trust objections and trigger phrases
- enrich the appraisal with social sentiment context
- feed content briefs and CTA copy

Operational loop:
**scan -> extract -> enrich appraisal -> create offer -> route -> measure -> repeat**

## Rule
Every surface must have a job.
- Blog = authority + lead gen + SEO/AEO + analytics
- Telegram = conversion + delivery + repeat use
- Compare service = lead routing + partner monetisation
- Pro / white-label = recurring B2B revenue
- Hit = signal extraction + copy + positioning + iteration

No inch should be decorative.
